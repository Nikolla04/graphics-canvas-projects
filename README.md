# **Computer Graphics Projects**

## **Overview**
This repository contains two distinct projects showcasing the fundamentals of computer graphics using **OpenGL** and the **HTML5 Canvas API**:

*   **[2D Static Graphic (HTML5 Canvas):](https://github.com/Nikolla04/graphics-canvas-projects/blob/main/html5-canvas/2d%20grb.html)** A clean, [static 2D vector scene](https://github.com/Nikolla04/graphics-canvas-projects/blob/main/html5-canvas/grb.JPG) rendered programmatically within the browser using canvas transformation matrices and custom rendering techniques.
*   **[2D Animated Scene (OpenGL):](https://github.com/Nikolla04/graphics-canvas-projects/blob/main/cpp/cpp/cpp.cpp)** A high-performance, hardware-accelerated 2D application featuring smooth time-based animations, custom fragment shaders, and real-time user interaction.[In the animation](https://github.com/Nikolla04/graphics-canvas-projects/blob/main/cpp/opengl.JPG), the ship moves, snowflakes fall, the windmill spins, and birds fly.
